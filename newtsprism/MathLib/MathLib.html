MathLib is a suite of object-oriented, advanced mathematical functions for use by any program in the Newton Operating System 2.x. Functions can easily be incorporated into any spreadsheet or programming application like QuickFigure or NS BASIC or even right into a word processor with nsScribe. Solve matrix equations, manipulate fractions, calculate pi, pick your lotto numbers, the possibilities are endless.

Now with version 3.00, a user interface!  MathLib has now been incorporated into Apple's Works Calculations program.  Check out the screen shots.  This is a mathematics interface that creates pages in Newton Works.  It was provided for free with the eMate.  Now various people have "released" it on the Web.  I can't in all conscience release someone else's work but can informally point you in the right direction to download the package.  Alternatively, you can find yourself an eMate and beam the package to your Newton.  Works Calculations works on the latest (and last) eMate and MessagePad model running NOS 2.1.  I use it on a MessagePad 2100.
