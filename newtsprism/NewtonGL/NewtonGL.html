NewtonGL is a 3D graphics language available to anyone who knows NewtonScript and OpenGL.  It started out as a school project then I decided to release it.  It is still in beta but is operational.  NewtonGL code can be run from NewtonWorks using nsScribe or can be compiled inside a program.
